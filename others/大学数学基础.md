## 大学数学基础

### 微积分

函数与极限

- 函数是对两个集合之间映射关系的描述
- 函数的常用性质是有界性、单调性、奇偶性、周期性
- 最常用的函数是初等函数，包括常函数、幂函数、指数函数、对数函数、三角与反三角函数
- 数列的极限是当序号趋于无穷时，数列的项收敛于某个常数
- 函数的极限是数列极限的推广，自变量趋于有限值或无穷时，映射得到的函数值收敛于某个常数
- 函数极限具有一些性质：
  - 唯一性：函数在某一点的极限值唯一
  - 局部有界性：在极限的邻域附近总是单调的
  - 局部保号性：在极限的邻域附近总是同号的
  - 数列极限推广：若一个数列极限存在，则对数列每一项施加相同的映射，函数值序列也存在极限，且极限必等于数列极限对应的函数值
- 极限的运算法则
  - 有限个无穷小的和是无穷小
  - 有界函数与无穷小的乘积是无穷小
  - 极限遵循四则运算
  - 数列的极限遵循四则运算
  - 复合函数的极限法则，从里向外依次求极限
- 函数极限的存在法则
  - 夹逼准则  g(x) <= f(x) <= h(x)
  - 柯西极限存在准则 |`xm - xn` | < k （k是任意正数，m , n > 一个正整数）
- 函数的连续性：左右极限存在且相等，且该点有定义
- 函数的间断点：不连续点（连续的否命题）

导数

- 导数是函数曲线切线斜率的极限
- 可导必须连续，连续不一定可导
- 导数的运算
  - 加减遵循四则运算
  - 乘法  [u(x)v(x)] = u'(x)v(x) + u(x)v'(x)
  - 除法  [u(x)/v(x)] = [u'(x)v(x) - u(x)v'(x)]/v^2(x)
- 反函数的导数是原函数导数的倒数
- 复合函数的求导遵循链式法则

微分

- 微分是函数值变化的极限（换句话说，微分是函数值一个极小的增量或减量）
- 微分的运算法则与导数的完全相同
- 微分的初衷和应用场景在于近似运算

微分中值定理

- 费马引理
- 罗尔定理
- 拉格朗日中值定理 （核心）：连续可导函数的一段区间内，总能找到一点的导数等于两端点连线的斜率值
- 柯西中值定理

导数的应用

- 区间极值：极值点对应的值可以有很多
- 区间最值：最值点对应的值只能有一个

不定积分

- 原函数：一组函数的导数如果都是同一个函数，则称这组函数是后者的原函数
- 区间上连续的函数必存在原函数
- 原函数存在必定可积
- 不定积分的结果需要加上常数 C，用以代表所有原函数组成的原函数簇
- 不定积分法则
  - 遵循加减法
  - 不定积分和求导在相差一个常数条件下是逆运算
- 不定积分的求解思路是换元积分法、分部积分法

定积分

- 若对一个有界函数在某区间内进行自变量的无限分割，其分隔间隔乘以对应小区间的端点的函数值的总和，如果存在极限，则该函数在这个区间上可积，且定积分值为极限值
- 定积分的性质
  - 遵循加减法
  - 遵循单调性，即函数值严格大的函数的积分也严格大
  - 常数因子可提出积分运算
- 定积分中值定理：总可以在区间内找到一点函数值，乘以区间长度等于该区间上的积分值

### 概率论与数理统计

随机事件

- 随机事件是某个随机现象在特定条件下所有可能结果的集合，每一个结果称为一个事件
- 在相同条件下重复N次某随机事件，其中某个结果出现的次数比上N就是该事件的概率
- 互斥事件的概率是可加的
- 条件概率是已知某事件发生条件下，另一事件的发生概率

随机变量与概率分布

- 随机变量是对随机事件的抽象，其每个取值对应一个可能结果
- 随机变量取值是离散的，称为离散随机变量，反之连续随机变量
- 离散随机变量的概率用分布列描述
- 连续随机变量的概率用概率密度函数描述
- 基本的离散随机变量的概率分布有
  - 二项分布/伯努利分布
  - 泊松分布
- 基本的连续随机变量的概率分布有
  - 均匀分布
  - 指数分布
  - 正太分布/高斯分布

贝叶斯定理

- 由条件概率的思想可以导出概率的乘法公式
- 如果一个事件B由一个事件集合A set决定，那么根据乘法公式的推广，可以用B关于`Ai`的条件概率表示B的总概率。这种思想称为全概率公式
- 贝叶斯定理基于上述两个基本想法，在确定了事件B发生的基础上，寻找导致B发生的`Ai`，这个过程叫作贝叶斯公式

随机变量的数字特征

- 数学期望：本质上是依据概率密度的加权平均值
- 方差：描述分布结果相对数学期望的离散程度

统计度量

- 大体上分为集中趋势度量和离散趋势度量、偏度以及封度
- 集中趋势度量
  - 算术平均数：最常用。分加权与不加权
  - 中位数：次常用
  - 众数：次常用
  - 四分位数
  - 调和平均数/倒数平均数：用于不知晓频数的数据
  - 几何平均数：用于数值为比率的数据
- 离散趋势度量
  - 极差/全距
  - 四分位数间距：上四分位数-下四分位数，常用
  - 方差：最常用
  - 标准差：方差的算术根
  - 变异系数/离散系数：用离散量比上算术平均数。最常用的是标准差比算术均值，称为标准差系数
- 偏度：最常用的是偏度系数，描述分布的偏斜程度
- 峰度：最常用的是峰度系数，描述分布峰的尖锐程度

参数估计

- 参数估计是用样本统计量来估计未知的总体参数
- 来源于样本的用于估计总体的统计量称为估计量，比如样本的算术平均值，样本方差
- 根据估计量推出的总体的统计量的值，称为估计值。估计值并不等于总体真正的统计量的真实值
- 点估计：根据样本，最终给出总体的某个统计量的一个估计值，是一个数
  - 矩估计法
  - 顺序统计量法
  - 极大似然估计：最常用
- 区间估计：根据样本，最终给出总体的某个统计量的值的范围，是一个区间，并附上置信度（一般是95%）

假设检验

- 假设检验是设定两个假设，一个叫作原假设，一个叫作备选假设。然后根据样本值对原假设进行判断，如果判断为真，则接受原假设，拒绝备选假设。反之接受备选假设，拒绝原假设
- 假设检验的本质是根据样本寻找足够证据去拒绝原假设，如果找不到，则接受
- 假设检验科学性的原理是，小概率事件在一次试验中几乎不可能发生
- 小概率事件设定得越极限，则发生的概率越低。那么一旦发生，就有很强的说服力表明是样本数据导致了该事件的发生，也就意味着原假设不成立
- 小概率事件的概率称为显著性水平，一般是0.05
- 假设检验的一般步骤
  - 提出原假设和备选假设
  - 构造检验统计量，在原假设成立的条件下，计算检验统计量的数学表达
  - 根据给定的显著性水平，确定拒绝域
  - 代入样本计算检验统计量的值
  - 根据结果是否落在拒绝域，绝对是否拒绝原假设
- 最常见的假设检验是估计正太分布的均值

> 字节的反转实验，本质上就是个假设检验。即先假设之前的实验效果是本次需求引起的，然后反转，去掉本次需求，如果实验效果消失了，说明确实是本次需求引起的
